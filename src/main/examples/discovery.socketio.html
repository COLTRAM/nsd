Discovery

Service type:


The blue button Get button uses NSD.getNetworkServices and requires the service type field to be an exact match to the service type, including the upnp: or zeroconf: prefix.

The orange button Get* button uses an extension NSDPlusPlus.discover which matches a substring of the service type. If you leave the service type empty, you get all discoverable services, whether UPnP or Bonjour.
